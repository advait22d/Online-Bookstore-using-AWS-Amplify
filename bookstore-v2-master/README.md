# 📚 Online Bookstore using AWS Amplify and Stripe

This project is a **fully functional, serverless online bookstore** built with **React** on the frontend and a suite of **AWS services** on the backend. It allows users to browse books, sign in securely, and make purchases using **Stripe**, while taking advantage of the scalability and flexibility of cloud-native architecture.

---

## 🚀 Features

- 🔐 **Authentication**: Secure user sign-up and sign-in using **Amazon Cognito**.
- 📦 **Inventory Management**: Dynamically fetch book listings using **GraphQL** with **AWS AppSync**.
- 🧾 **Payments Integration**: Integrated **Stripe** for seamless and secure checkout.
- 🗃️ **Serverless Backend**: Utilized **Lambda functions** for handling backend logic like order processing.
- 💾 **Data Storage**: Orders and user data stored in **Amazon DynamoDB**.
- 🖼️ **Media Hosting**: Book images and assets hosted on **Amazon S3**.
- 🌍 **Global Hosting**: Deployed with **AWS Amplify** for continuous integration and deployment.

---

## 🛠️ Tech Stack

- **Frontend**: React, TailwindCSS
- **Backend**: AWS AppSync (GraphQL), Lambda
- **Authentication**: Amazon Cognito
- **Database**: Amazon DynamoDB
- **Media**: Amazon S3
- **Payments**: Stripe
- **Hosting/CI**: AWS Amplify
